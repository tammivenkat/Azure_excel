## Network Security Group Rules

| Priority | Protocol | Port | Action |
|----------|----------|------|--------|
| 100 | TCP | 80 | Allow HTTP |
| 101 | TCP | 22 | Allow SSH |

### Purpose
- HTTP (80): Allows users to access the web application.
- SSH (22): Allows administrators to securely connect to the Linux VM.
