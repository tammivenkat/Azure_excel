## Lessons Learned

- Azure resources are organized using Resource Groups.
- Regions determine geographical deployment.
- Availability Zones improve High Availability.
- Every Azure resource belongs to a Resource Group.

---

## Common Mistakes

- Creating resources in the wrong Region.
- Deleting the Resource Group accidentally.
- Naming Resource Groups inconsistently.
